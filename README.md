Project Description
In this project, we worked with Debian 13 on a virtual machine to
learn about security in Linux systems. We installed the system
using full disk encryption with LUKS and LVM to protect data. We 
also used KeePassXC to manage secure passwords and 
understand entropy. In addition, we explored hash functions and 
performed attacks using Hashcat to observe how weak passwords can be
compromised. Finally, we applied security concepts in email communication 
and compromised. Finally, we applied security concepts in email communication
Objectives
Learn how to install a Linux system with full disk encryption to protect data
Understand how secure passwords and entropy work.
Analyze the use of hash functions and their importance in security.
Apply tools like GPG and Git to improve security and authenticity.
